# Array.prototype.includes()

判断一个数组是否包含一个指定的值,返回布尔值;

参数:

  • valueToFind -- 需要查找的元素值
  • fromIndx(可选) -- 从哪个索引值开始查找
var a = [1,2,3,4,5,6]

a.includes(2)  // true
a.includes(2,3)  // false
a.includes(5,-2)  // true
a.includes(5,-1)  // false

# Array.prototype.indexOf()

返回指定元素在数组中的第一个索引,如果不存在,返回-1;

参数:

  • searchElement -- 需要查找的元素
  • fromIndx(可选) -- 从哪个索引值开始查找
var array = [2, 5, 9];

array.indexOf(2);     // 0
array.indexOf(7);     // -1
array.indexOf(9, 2);  // 2
array.indexOf(2, -1); // -1
array.indexOf(2, -3); // 0

# Array.prototype.lastIndexOf()

返回指定元素在数组中的最后一个索引,如果不存在,返回-1,从后往前查找,从fromIndex处开始

参数:

  • searchElement -- 需要查找的元素
  • fromIndx(可选) -- 从哪个索引值开始查找
var array = [2, 5, 9, 2];

array.lastIndexOf(2);      // 3
array.lastIndexOf(7);      // -1
array.lastIndexOf(2, 3);   // 3
array.lastIndexOf(2, 2);   // 0
array.lastIndexOf(2, -2);  // 0
array.lastIndexOf(2, -1);  // 3

# Array.prototype.some()

测试数组中是否至少有一个元素通过指定函数测试,返回布尔值

语法:


arr.some(callback(element[,index[,array])[,thisArg])

参数:

  • callback -- 用于测试每个元素的函数
  • thisArg(可选) -- 执行callback时使用的 this

[2, 5, 8, 1, 4].some(x => x > 10);  // false
[12, 5, 8, 1, 4].some(x => x > 10); // true

# Array.prototype.every()

测试数组中是否所有元素都能通过指定函数测试,返回布尔值,其他同上


[2, 5, 8, 1, 4].some(x => x > 10);  // false
[12, 15, 18, 11, 14].some(x => x > 10); // true

# Array.prototype.filter()

创建一个新数组,包含原数组中通过指定函数测试的所有元素

语法:


arr.filter(callback(element[,index[,array])[,thisArg])

参数:

  • callback -- 用于测试每个元素的函数
  • thisArg(可选) -- 执行callback时使用的 this

let filtered = [12, 5, 8, 130, 35].filter(element => element >= 10);
// filtered is [12, 130, 35]

# Array.prototype.find()

返回数组中满足指定函数的第一个元素的值,没有符合的返回undefined


const inventory = [
    {name: 'apples', quantity: 2},
    {name: 'bananas', quantity: 0},
    {name: 'orange', quantity: 5}
];

let findOrange = inventory.find(element => element.name === 'orange')

// { name: 'orange', quantity: 5 }
